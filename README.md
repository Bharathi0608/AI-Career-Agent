# AI Career Agent

AI-powered Resume Analysis System built using:

- Python
- Flask
- LangChain
- LangGraph
- Groq LLM

## Features

- Resume Skill Extraction
- Job Recommendations
- Resume Scoring
- Interview Question Generation
- Cover Letter Generation
- Multi-Agent Workflow using LangGraph
